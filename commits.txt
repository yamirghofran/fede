Commit 5 — feat(retrieval): add ScriptRetriever with hierarchical scene/sentence search
Create vector_db/retrieval.py
search_sentences(query_embedding, top_k, movie_id_filter?) → returns List[SentenceResult] with scores and full payload
search_scenes(query_embedding, top_k, movie_id_filter?) → returns List[SceneResult]
hierarchical_search(query_embedding, top_k) — searches sentences, deduplicates to parent scenes via scene_id metadata, merges with direct scene search results, re-ranks by max sentence score per scene, returns top-k unique scenes
All methods accept a pre-computed embedding vector (decoupled from the embedding model)
